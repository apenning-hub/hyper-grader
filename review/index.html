Testing Guide

1. First Launch

1
Launch the app. A 3-page onboarding walkthrough appears. Click Next through all pages, then Get Started.

2. Create a Course & Assessment

2
Click the + button in the sidebar to create a course. Click the + next to the course name to add an assessment.

3. Import a Rubric

3
Go to File → Import Rubric… (⇧⌘I) and select the SampleRubric.csv you downloaded above. The rubric loads with 4 criteria.

4. Grade a Student

4
Type a student name, click rating pills for each criterion, optionally record voice notes via the microphone button, then click Generate to produce AI feedback.
AI Model: On-device AI requires macOS 15.1+ with Apple Intelligence enabled. Alternatively, enter a Claude API key in Preferences (⌘,). The app is fully functional for rubric import, grading, and export without AI.

5. Navigate & Export

5
Use ⌘→ / ⌘← to move between students. Click Export CSV in the toolbar to export grades.

6. Import/Export Library

6
File → Export Library… (⇧⌘E) saves all data as a .hyper file. File → Import Data… (⇧⌘O) restores from a .hyper or .json file. You can test with the SampleData.hyper file above.

Rubric CSV Format

Each row defines one criterion. Columns repeat in groups of 3 for each rating level:

Column Content
1Rubric name
2Criterion name
3Criterion description
4(empty)
5, 6, 7Rating name, description, points
8, 9, 10Next rating name, description, points
…Repeat for additional rating levels

The sample CSV includes 4 criteria with 5 rating levels each (Excellent through Unsatisfactory).

All Features

  • Course and assessment organization
  • Rubric import from CSV
  • Voice note recording with on-device transcription
  • AI feedback generation (on-device Apple Intelligence or Claude API)
  • Student navigation and grading history
  • Batch student queue management
  • Student performance profiles and trajectory tracking
  • Assessment statistics (mean, median, per-criterion)
  • Comparative grading (side-by-side student comparison)
  • Feedback consistency checker
  • Feedback customization (tone, length, format, focus, voice)
  • Tone calibration sliders
  • CSV grade export
  • Library export/import (.hyper and .json)
  • Keyboard shortcuts throughout

Privacy & Data

  • No account or login required
  • On-device AI mode: no data leaves the Mac
  • API key stored in macOS Keychain
  • Student names anonymized in AI prompts
  • Speech transcription is on-device only
  • Microphone access requested at first recording
