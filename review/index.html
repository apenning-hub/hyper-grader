Testing Guide

1. First Launch & Onboarding

1
Launch the app. A 3-page onboarding walkthrough appears. Click Next through all pages, then Get Started.

2. Create a Course & Assessment

2
Click the + button in the sidebar to create a course (e.g. “English 101”). Click the + next to the course name to add an assessment (e.g. “Essay 1”). The sidebar shows the course/assessment hierarchy.

3. Import a Rubric

3
Go to File → Import Rubric… (⇧⌘I) and select the SampleRubric.csv. The rubric loads with 4 criteria and 5 rating levels each. A green checkmark appears in the toolbar.

4. Import a Student List

4
Click the clipboard icon (Student Queue) in the student bar. Click Import File… and select SampleStudentList.csv. 10 student names appear in the preview. Click Save. A progress bar appears showing “0 of 10”.

5. Grade a Student

5
The first student name from the queue auto-fills. Click rating pills for each criterion (color-coded: green = high, red = low). Optionally click Record Voice Notes to add spoken observations (requires microphone permission). Click Generate to produce AI feedback.
AI Model: On-device AI requires macOS 16.0+ with Apple Intelligence enabled. Alternatively, enter a Claude API key in Preferences (⌘,). The app is fully functional for rubric import, grading, and export without AI — simply type feedback manually.

6. Navigate Between Students

6
Use ⌘→ / ⌘← to move between students. The student dropdown (person icon) shows grading history with scores. With a batch queue active, Next Student auto-advances to the next ungraded student.

7. Customize Feedback

7
Click the slider icon in the assessment bar to open Feedback Instructions. Configure tone (Encouraging/Balanced/Direct/Rigorous), length, format, focus areas, and voice. Add custom instructions. These are saved per-assessment.

8. Export Grades

8
Click the share icon in the assessment bar to Export CSV. The exported file includes student names, per-criterion ratings, total scores, percentages, generated feedback, and voice transcripts.

9. View Statistics & Compare Students

9
Click the chart icon for Assessment Statistics (mean, median, per-criterion averages). Click the people icon to Compare Students side-by-side. Click the shield icon for a Feedback Consistency check. These require at least 2 graded students.

10. Student Profiles & Trajectory

10
Click the profile card icon to view a student’s performance across all assessments in the course, including trend analysis and score history. Toggle the trajectory icon to include this context in AI prompts.

11. Import/Export Library

11
File → Export Library… (⇧⌘E) saves all data as a .hyper file. File → Import Data… (⇧⌘O) restores from a .hyper or .json file. A preview sheet shows the summary and lets you choose Merge or Replace. Test with the SampleData.hyper file above.

12. Assessment Locking

12
Right-click an assessment in the sidebar and select Lock Assessment. The grading view becomes read-only with a lock banner. Unlock from the banner or sidebar context menu.

File Format Reference

Rubric CSV

Row 1 is a header. Each subsequent row defines one criterion:

ColumnContent
1Rubric name
2Criterion name
3Criterion description
4(empty)
5, 6, 7Rating name, description, points
8, 9, 10Next rating… (repeat for each level)

Compatible with Canvas LMS rubric CSV exports.

Student List CSV

A CSV with a header containing “name” or “student”. Only the name column is used; other columns are ignored. Alternatively, a plain text file with one name per line.

Library File (.hyper)

A JSON envelope containing all courses, assessments, rubrics, and grading records. Created via File → Export Library. Can also import bare JSON arrays for backwards compatibility.

All Features

  • Course and assessment organization with sidebar navigation
  • Rubric import from Canvas-compatible CSV
  • Voice note recording with on-device speech transcription
  • AI feedback generation (on-device Apple Intelligence or Claude API)
  • Feedback customization: tone, length, format, focus areas, voice
  • Tone calibration sliders (encouragement, specificity, verbosity)
  • Feedback refinement and regeneration with custom prompts
  • Batch student queue from CSV or text file
  • Student navigation with grading history and scores
  • Student performance profiles and trajectory tracking across assessments
  • Assessment statistics (mean, median, min/max, per-criterion averages)
  • Comparative grading (side-by-side student comparison)
  • AI-powered feedback consistency checker
  • Assessment locking (read-only mode)
  • CSV grade export with scores, feedback, and transcripts
  • Library export/import (.hyper and .json) with merge/replace options
  • Finder file association for .hyper files
  • Keyboard shortcuts throughout

Privacy & Data

  • No account or login required
  • On-device AI mode: no data leaves the Mac
  • Claude API key stored securely in macOS Keychain
  • Student names anonymized in all AI prompts
  • Speech transcription is fully on-device
  • Microphone permission requested only at first recording
  • All data stored locally; no telemetry or analytics
